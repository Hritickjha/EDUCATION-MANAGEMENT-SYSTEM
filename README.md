# Education Management System

## Overview
The Education Management System is designed to streamline and manage various educational processes, including student enrollment, course management, and grading.

## Features
- **Student Enrollment**: Manage student registration and enrollment.
- **Course Management**: Create and manage courses.
- **Grading System**: Track and record student grades.

## Installation
1. Clone the repository:
    ```bash
    git clone https://github.com/Hritickjha/EDUCATION-MANAGEMENT-SYSTEM.git
    ```
2. Navigate to the project directory:
    ```bash
    cd EDUCATION-MANAGEMENT-SYSTEM
    ```

## Usage
1. Run the main script to start the Education Management System:
    ```bash
    python main.py
    ```
